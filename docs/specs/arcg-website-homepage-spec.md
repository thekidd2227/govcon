# ARCG Website — Homepage Spec

**Subject:** Public homepage for Ariel's River Contracting Group, LLC.
**Positioning:** Premium, government-ready SDVOSB execution partner. McKinsey-lite / SoHo / defense-adjacent. Not a federal-template clone.
**Scope of this document:** Homepage only. Subpages referenced where needed.

---

## 0. Pre-launch flag — domain/brand mismatch

Legal name is **Ariel's River Contracting Group, LLC** (brand: *Ariel's River* / *ARCG*). Domain is **arivergroup.com** and email is **info@arivergroup.com** — both read as "A River Group," which the brief explicitly prohibits as a display name.

Recommended path:

1. Acquire **arielsriver.com** or **arielsrivercontracting.com** (or **arcg-llc.com**), 301 redirect arivergroup.com, migrate email. Cleanest.
2. Keep arivergroup.com; add a footer line: *"arivergroup.com is the registered web domain of Ariel's River Contracting Group, LLC."* Mitigation only.
3. Accept the friction.

The spec below uses arivergroup.com verbatim with the mitigating footer line. Remove if option 1 or 3.

## 8. Procurement-readiness corrections

1. **Domain/brand alignment** — see §0. Most important.
2. **Capability Statement PDF must mirror the site exactly** — same legal name, identifiers, NAICS order, primary highlighted (561710), same POC.
3. **SAM.gov public profile** must list arivergroup.com *and* the legal name verbatim. Same for DSBS.
4. **SDVOSB verification language** — once VA SBA verification is live, link directly to the verification page from Section 4.
5. **Phone format standardization** — pick one (`347.285.5750` or `(347) 285-5750`) and use it everywhere.
6. **Add a named-POC email** — `jmc@arivergroup.com` alongside info@ so federal buyers can route directly.
7. **Past Performance depth** — disclose scope (months, value range if releasable, NAICS used). One reference is acceptable to start; thin pages quietly lose credibility.
8. **Brand mark / wordmark** — develop a typographic wordmark in Fraunces. Single SVG, no icon needed.
9. **Capabilities depth pages** — each `/capabilities/<lane>` answers: scope · geographic coverage · response SLA · sample customers · lane-specific certifications.
10. **Privacy + Accessibility pages** — `/privacy` and `/accessibility` (with Section 508 conformance statement) are expected on government-facing sites.
