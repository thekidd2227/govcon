# Daily Fast-Cash Capture Loop

## Morning

1. Run Fast-Cash Scan.
2. Review top 10 ranked actions.
3. Kill bad fits immediately.
4. Move quoteable work to Sub Sourcing or Quote Ready.
5. Send safe outreach only where outreach status is green.

## Midday

1. Request sub/vendor quotes.
2. Verify insurance/license/bonding.
3. Build micro quote packets.
4. Submit through official channel only.

## End of day

1. Export daily brief.
2. Log opportunity decisions.
3. Log outreach.
4. Set next follow-ups.
