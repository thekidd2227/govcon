# Charlie Fast-Cash GovCon Operating Model

## Objective

Land feet first in federal work by targeting micro-purchases, simplified acquisition opportunities, and small contracts under a conservative $250K ceiling.

## Core model

ARCG acts as prime contractor and retains meaningful management, administration, scheduling, QA/QC, compliance documentation, client communication, invoicing, subcontractor coordination, issue escalation, and final accountability.

Subcontractors or vendors may perform field execution when legally and operationally appropriate.

## Target profile

- Certifications: SDVOSB, HUBZone, MBE, DBE, SBE
- Preferred ceiling: $250K
- Minimum target net margin: 25%
- Preferred response speed: 24–72 hours
- Preferred model: subcontractor-supported, prime-managed
- Discovery: any NAICS allowed, but every lane must be labeled and risk-scored

## Hard avoids

- clearance-heavy work
- daily/weekly prime physical onsite requirement
- construction-heavy work unless small, simple, and sub-covered
- bonding-heavy work without verified bonding path
- license-heavy work without verified licensed sub
- equipment-heavy/supply-only work without compliant reseller/nonmanufacturer path
- complex multi-volume RFPs
- low-value work with high admin burden
